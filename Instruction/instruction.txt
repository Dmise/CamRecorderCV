|----------------------|
|*** Необходимое ПО ***|
|----------------------|

1) Устанавливаем ffmpeg
https://www.wikihow.com/Install-FFmpeg-on-Windows

|---------------------------|
|*** Регистрация Сервиса ***|
|---------------------------|
* 1)Содержимое архива release.1.0.zip распаковываем в папку C:\Program Files\Video Record Service\   

* 2)Полсе разархивации регистрируем сервис в системе командой (копируем следующую строчку в cmd или PowerShell):
sc.exe create "Video Record Service" binpath="C:\Program Files\Video Record Service\VideoRecordService.exe"

если все ок. получаем в консоле: [SC] CreateService SUCCESS

* 3)настраиваем сервис на автозапуск.
- в поиске windows ищем Services
- в списке сервисов крутим до "Video Record Service" Правой кнопкой. ->  Properties ->  в выпдающем списке Startup type = Automatic - > Жмем ОК

* 4) Либо перезагружаем компьютер и нанется запись. 
- Либо запускаем уже сейчас сервис в ручную. Все в том же списке сервисов правой кнопкой "Video Record Service" - > Start

|-----------------------------------------|
|*** Настрйока конфигурационного файла ***|
|-----------------------------------------|
* 1) Длительность дежурного видео указывается в минутах. в конфигурационном файле appsettings.json
"VideoLength": 60

* 2) в файле appsettings устанавливаем адресс сетевой папки в которую хотим сохранять дежурное видео. Заменяем значение по умолчанию на желаемое

пример
было: saveto": "C:\\Program Files\\Video Record Service\\Videos\\"  
стало: saveto": "\\\\DESKTOP-V7Q9VJC\\SharedFolder\\"   (символ бэкслеша экранируется - получается двойной бэкслеш для разделения папок. и 4ой бэкслеш для начала адреса сетевой папки)

* 3) Определяем название камеры
командой в командой строке: ffmpeg -list_devices true -f dshow -i dummy 
После чего будет длинный вывод команды, ищем стрчоку с "HP HD Camera" (video)  (в моем примере это HP HD Camera)

* 4) Устанавливаем назваине камеры в конфигурационном файле
"CameraName": "HP HD Camera",   


|-------------------------------|
|*** Если возникли сложности ***|
|-------------------------------|

Удалить сервис если что-то пошло не так можно командой из powershell или cmd.
sc.exe delete "Video Record Service"
